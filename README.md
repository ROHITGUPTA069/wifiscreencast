# wifiscreencast

An Android application that allows users to cast their mobile screen to a laptop, PC, or Smart TV over Wi-Fi using the RTSP protocol.

Created by: Rohit Gupta

Features
Splash screen with “Rohit Gupta”

Main screen displays app title and controls

Start/Stop screen casting over Wi-Fi

Shows device local IP and RTSP streaming URL

Works on laptops, PCs, and Smart TVs that support RTSP
